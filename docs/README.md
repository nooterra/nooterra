## Core Concepts

- **[Approvals](/approvals)** — Human-in-the-loop flows. Workers pause on sensitive actions and route to you with full context.
- **[Scheduling](/scheduling)** — Continuous, cron, or webhook-triggered. Workers run 24/7.
- **[AI Providers](/providers)** — ChatGPT, Claude, Gemini, Groq, OpenRouter. Swap anytime.
- **[Performance](/performance)** — Track worker output, cost, and approval patterns over time.
