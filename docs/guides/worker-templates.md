# Worker Templates

Templates are pre-configured workers for common business tasks. Select a template, review the charter, and activate — no description needed.

## Available Templates

| Template | What it does | Schedule |
|---|---|---|
| **Customer Support** | Handle inbound questions, look up accounts, draft replies | Continuous |
| **Inbox Triage** | Read email, categorize by urgency, forward to the right person | Every 15 min |
| **Price Monitor** | Track competitor pricing and alert on changes | Hourly |
| **Social Monitor** | Watch for brand mentions across the web | Every 2 hours |
| **PR Reviewer** | Review GitHub pull requests for code quality | Every 30 min |
| **Meeting Summary** | Summarize discussions and extract action items | Weekdays 10 AM |

## Using a template

1. From the dashboard, click **Create** and switch to **Worker** mode
2. Scroll down to the template section
3. Click a template to preview its charter
4. Edit any permissions if needed
5. Click **Activate**

## Customizing templates

Templates are starting points. After activation, you can edit the worker's charter to:

- Add or remove canDo, askFirst, and neverDo rules
- Change the schedule
- Switch the AI model
- Add integrations (Gmail, Slack, Stripe, etc.)
