# Roadmap

This roadmap reflects the current Trust OS direction.

## Phase 1: Production Core (current)

- Deterministic policy runtime enforcement
- Execution binding + deterministic receipts
- Dispute/reversal state handling
- Operator controls and audit traces
- Terminal-first onboarding for supported hosts

## Phase 2: Frictionless Adoption

- Faster first verified receipt path
- Better managed wallet default flow
- One-command onboarding quality across host targets
- Reliability and operator visibility improvements

## Phase 3: Platform Expansion

- Stronger tenant automation and enterprise controls
- Richer profile/policy simulation workflows
- Expanded adapter surface under same trust contract

## Phase 4: Agentverse Infrastructure

- Cross-runtime trust fabric
- Stronger inter-agent settlement primitives
- Shared controls ecosystem and federation patterns
