# Agent Prompt Templates

These templates are drop-in prompts for specialized agents.
Use one per task, with strict ownership boundaries from `planning/ownership/agent-roster.md`.

## Included templates

- `ceo-orchestrator.md`
- `kernel-protocol-agent.md`
- `api-control-agent.md`
- `provider-ecosystem-agent.md`
- `mcp-integration-agent.md`
- `reliability-devops-agent.md`
- `qa-security-agent.md`
- `gtm-docs-agent.md`

## Required output shape for all agents

- `Summary`
- `Changes`
- `Validation`
- `Risks`
- `Next handoff`
